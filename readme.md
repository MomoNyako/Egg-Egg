# Eat Egg

Eat Eggs 的非官方移植版，为 Minecraft 添加多种蛋类食物。

## 物品

| 物品 | 饥饿值 | 饱和度 | 特殊效果 |
|------|--------|--------|----------|
| 熟鸡蛋 (Cooked Egg) | 5 | 0.5 | - |
| 煎蛋卷 (Omelette) | 6 | 1.26 | 40% 概率获得 10s 夜视 |
| 培根鸡蛋 (Bacon Eggs) | 8 | 1.6 | 50% 概率获得 10s 生命恢复 |
| 英式全餐 (Full English Breakfast) | 10 | 2.0 | 10s 夜视 + 缓慢 + 生命恢复，40s 力量 + 抗性 |

## 合成配方

- **熟鸡蛋**：鸡蛋 → 熔炉 / 营火 / 烟熏炉烧制
- **煎蛋卷**：3× 熟鸡蛋（无序合成）
- **培根鸡蛋**：熟猪排 + 2× 熟鸡蛋（无序合成）
- **英式全餐**：熟猪排 + 熟鸡蛋 + 熟牛肉 + 面包 + 棕色蘑菇 + 烤土豆（无序合成）

## 额外特性

模组的食物可用于喂食已驯服的狼（回血、催长幼狼、进入繁殖状态）。
